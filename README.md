## Functions

SkinnyViewIndex
 : open index.html
SkinnyViewEdit
 : open edit.html
SkinnyViewNew
 : open new.html
SkinnyViewShow
 : open show.html
SkinnyListModels
 : List All models unser /model

### List Model page option
After SkinnyListModels, you can open model, controller.
|m| open model|
|c| open controller|
